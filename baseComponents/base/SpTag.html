# 参数列表

参数 说明 类型 必填 可选值 默认值
limit 标签显示类型 Boolean 否 true 绑定列表，有选中和非选中操作 / false 数据无限，可添加删除，字符串数组 true
options 传入的数据,其中约定数组对象为：{label:"标签 1"，value:"1"，checked:true}；label 为文本描述，value 为属性值，checked 为是否选中的状态 Array 否 - []
optionValue 属性值对应的配置字段名称 String 否 - value
optionLabel 文本描述对应的配置字段名称 String 否 - label

# 事件

参数 说明 回掉参数
handleClick 选中或取消时会触发 当前选项值index
handleRemove 移除项时会触发 移除的选项值item
handleAdd 添加项时会触发 添加的值newTag
