# 参数列表

参数 说明 类型 必填 可选值 默认值
value multSelector绑定值，建议使用value,便于最终结果遍历 String 是 - -
navOptions 导航绑定数据 如[{key: "hot",text: "热门城市"}] key为点选值 text为显示名称 Arry 是 - -
itemOptions 导航选项对应下方数据 Object 是 - -
placeholder placeholder提示语 String 否 - -
size 输入框大小，提供三种尺寸 big:380px middle:270px small:120px String 否 big / middle / small middle
optionValue 属性值对应的配置字段名称 String / Number 否 - value
optionLabel 文本描述对应的配置字段名称 String 否 - label

# 事件

参数 说明 回调参数
changeSelectItem 导航绑定选项切换 当前导航对应值
valueChange 当行下对应数据选择切换 当前选择项
