# 参数列表

参数 说明 类型 必填 可选值 默认值
options 传入的数据 Array 是 - -
optionValue 属性值对应的配置字段名称 String 否 - value
optionLabel 文本描述对应的配置字段名称 String 否 - label
childrenName 子节点配置字段字段名 String 否 - children
defaultId 默认选中的节点 String,Number 否 - -

changeId 展开节点时触发 当前节点id
