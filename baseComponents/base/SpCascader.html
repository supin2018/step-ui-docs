# 参数列表

参数 说明 类型 必填 可选值 默认值
values SpCascader绑定值，建议使用values,便于最终结果遍历 String 是 - []
placeholder placeholder提示语 String 否 - 请选择
size 输入框大小，提供三种尺寸 big:380px middle:270px small:120px String 否 big / middle / small middle
showAllLevel 选中回显显示数据层级，true 为显示三级，false 为显示最后一级 Boolean 否 true / false true
separator 选中回显显示数据分隔符 String 否 - /
props 属性配置，默认key值为{value:"",label:"",children:[]},可通过此属性配置关联属性，如{value: "id", label: "discribe", children: "children"} Object 否 - -
options 列表数据,其中约定数组对象为{label:"北京"，value:"1",children:[]} label为文本描述，value为属性值，children 为下一级数据 。如果想更改绑定值，参考props属性 Array 否 - []
optionValue 属性值对应的配置字段名称 String / Number 否 - value
optionLabel 文本描述对应的配置字段名称 String 否 - label
disabled 是否禁用 Boolean 否 true / false false
valueSearch 级联第一级搜索框 Boolean 否 true / false false
validateEvent 是否需要验证 Boolean 否 true / false true

# 事件

参数 说明 回调参数
valueChange 输入数据发生变化 返回的是选框绑定值
handleClose 级联下拉框 收起时候触发 -
searchChange 搜索输入框输入时触发 仅当valueSearch参数为true时生效 返回的是实时输入内容
handleSearch 搜索输入框点击搜索时触发 仅当valueSearch参数为true时生效 返回的是搜索框输入内容
