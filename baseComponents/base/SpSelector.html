# 参数列表

参数 说明 类型 必填 可选值 默认值
value seletor绑定值，建议使用value,便于最终结果遍历 String 是 - null
placeholder placeholder 提示语 String 否 - 请选择
disabled 是否禁用 Boolean 否 true / false false
size 输入框大小，有三种尺寸供选择 big:380px middle:270px small:120px String 否 big / middle / small middle
options 列表数据,其中约定数组对象为{label:"北京"，value:"1"} label为文本描述，value为属性值，如果没有属性值，则属性值默认为文本描述 Array 否 - []
name 原生name 属性 String 否 - -
id 原生id属性 String 否 - -
optionValue 属性值对应的配置字段名称 String / Number 否 - value
optionLabel 文本描述对应的配置字段名称 String 否 - label
showDefault 是否添加“不限”选项，对应的value值为空字符串 Boolean 否 true / false false
editable 选框是否可编辑 Boolean 否 true / false false

# 事件

参数 说明 回掉参数
valueChange 选择发生变化时触发 返回的是选框绑定值
